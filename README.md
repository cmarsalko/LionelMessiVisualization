# Lionel Messi's Shot Data Visualization
By Chase Marsalko
In this project, I created eight visualizations analyzing Lionel Messi's career shot data. Each visualization was chosen to highlight a unique perspective on Messi's legendary career, covering his time at FC Barcelona, Paris Saint-Germain, and with the Argentina national team.
The dataset used is the free StatsBomb Open Data, covering 34 competitions across Messi's career and totaling 2,632 shots with x/y pitch coordinates, shot outcomes, techniques, and expected goals (xG) values. All visualizations were created in Python using statsbombpy, mplsoccer, matplotlib, pandas, and wordcloud.
